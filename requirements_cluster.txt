# Requirements for HPC Cluster (Param Shakti) - Local LLM + Vision
# ================================================================
# For running run_mas_diagnosis.py with ACTIVE_PROVIDER=local
#
# INSTALL ORDER (CRITICAL):
#   1. Create conda env with Python 3.10:
#      conda create -n medsam python=3.10 -y && conda activate medsam
#
#   2. Install PyTorch + torchvision FIRST (match your CUDA version):
#      pip install torch torchvision --index-url https://download.pytorch.org/whl/cu121
#      NOTE: torchvision is required by segment_anything (MedSAM). Without it you get
#            "No module named 'torchvision'" at runtime.
#
#   3. Install MedSAM from GitHub (bowang-lab):
#      git clone https://github.com/bowang-lab/MedSAM
#      cd MedSAM && pip install -e .
#      cd ..
#
#   4. Download MedSAM checkpoint:
#      bash download_ckpts.sh
#      # Or manually: wget https://huggingface.co/wanglab/medsam-vit-b/resolve/main/medsam_vit_b.pth -P ./checkpoints/
#
#   5. Install remaining dependencies:
#      pip install -r requirements_cluster.txt
#
# Reference:
#   Ma et al., "Segment Anything in Medical Images", Nature Communications 2024
#   https://github.com/bowang-lab/MedSAM
# ================================================================
